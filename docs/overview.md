# IlmAI — Overview

IlmAI is a domain-specific AI assistant focused on Islamic knowledge for users in Bosnia and Herzegovina and the wider Balkan region. It uses Retrieval-Augmented Generation (RAG) to provide source-grounded answers from Qur'an, Hadith, and Tafsir texts.

## Key Differentiators

- **Domain-specific**: Not a general chatbot — focused on Islamic knowledge
- **Source-grounded**: Every answer cites its sources (ayah, hadith, tafsir)
- **Bilingual**: Supports Bosnian and English
- **RAG-powered**: Uses vector search for semantic retrieval
- **Transparent**: Users can verify answers against original sources
